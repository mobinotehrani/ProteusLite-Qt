# بخش چهارم — مدیریت کتابخانه قطعات

این بسته نسخه تجمعی بخش‌های ۱ تا ۴ است. بخش چهارم از نو و بر اساس رفتار نسخه اصلی پروژه طراحی شده و فایل‌های مربوط به بخش‌های ۱ تا ۳ بدون تغییر منطقی نگه داشته شده‌اند.

## قابلیت‌های بخش چهارم

- نمایش قطعات در دسته‌های ساختاریافته با `QTreeWidget`
- جست‌وجوی لحظه‌ای روی نام، دسته، توضیح، شناسه و مقدار پیش‌فرض
- پیش‌نمایش شماتیک قطعه انتخاب‌شده
- نمایش خلاصه نام، دسته و مقدار پیش‌فرض قطعه
- افزودن قطعه به فهرست Active Project Parts
- جلوگیری از اضافه‌شدن قطعه تکراری
- حذف قطعه از فهرست فعال
- آماده‌سازی Place از کتابخانه یا فهرست فعال
- نمایش وضعیت عملیات در Status Bar
- حفظ Grid، Snap، Zoom، Pan و مختصات بخش سوم

## طراحی چیدمان

چیدمان پنل عمداً ساده و نزدیک به نسخه اصلی پروژه نگه داشته شده است. Tree View و Active List هر دو دارای سهم قابل تغییر از ارتفاع پنل هستند و Preview ارتفاع محدود دارد؛ بنابراین در پنجره‌های معمولی، بخش Active Project Parts و دکمه‌های آن بیرون صفحه قرار نمی‌گیرند.

## مرز بخش چهارم

در این مرحله با انتخاب Place، نام قطعه و مختصات Snap‌شده روی بوم آماده می‌شود. ایجاد شیء گرافیکی واقعی قطعه، انتخاب، Drag & Drop، Rotate، Mirror، Delete و Properties در بخش پنجم انجام می‌شود.

## تست‌های ضروری

1. باز و بسته شدن دسته‌ها
2. جست‌وجوی `resistor` و بازگشت فهرست پس از پاک‌کردن جست‌وجو
3. تغییر Preview با انتخاب قطعات مختلف
4. افزودن Resistor به Active Project Parts
5. جلوگیری از افزودن تکراری Resistor
6. مشاهده کامل Active List و دکمه‌های Place Active و Remove Active
7. حذف قطعه فعال
8. آماده‌سازی Place پس از ایجاد یا بازکردن پروژه
9. لغو Place با Escape یا راست‌کلیک
10. اجرای بدون خطای قابلیت‌های بخش‌های ۱ تا ۳
