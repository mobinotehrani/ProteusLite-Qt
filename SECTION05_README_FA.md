# بخش پنجم: مدل قطعه و سیستم اتصال اولیه

این نسخه ادامه مستقیم بخش چهارم است. فایل‌های کدنویسی بخش‌های ۱ تا ۴ بدون تغییر نگه داشته شده‌اند و قابلیت‌های بخش پنجم با فایل‌های جدید به پروژه افزوده شده‌اند. تنها فایل قبلی که تغییر کرده `CMakeLists.txt` است تا فایل‌های جدید در Build وارد شوند.

## مسئول Branch و Push

مبین اسکاره طهرانی

Branch پیشنهادی:

`feature/05-component-model-wiring`

Commit پیشنهادی:

`feat: add component model and 90-degree wiring`

## قابلیت‌های افزوده‌شده

- کلاس گرافیکی و داده‌ای `ComponentItem`
- قرار دادن واقعی قطعه روی بوم پس از انتخاب از کتابخانه
- انتخاب تکی و چندگانه با امکانات خود `QGraphicsScene`
- Drag & Drop همراه با Snap
- چرخش ۹۰ درجه با کلید `R` یا منوی راست‌کلیک
- قرینه‌سازی افقی و عمودی با کلیدهای `H` و `V`
- پنجره Properties با دبل‌کلیک
- حذف قطعه با Delete یا منوی راست‌کلیک
- مدل‌های `PinModel`، `WireModel`، `NodeModel` و `JunctionModel`
- تشخیص پایه هنگام کلیک نزدیک Pin
- رسم سیم شکسته با زاویه‌های ۹۰ درجه
- تغییر پویا مسیر سیم هنگام جابه‌جایی، چرخش یا قرینه‌سازی قطعه
- ساخت Junction با اتصال سیم جدید به سیم قبلی یا گزینه Add Junction Here
- انتخاب و حذف مستقل سیم‌ها و Junctionها

## روش استفاده

1. یک پروژه جدید بسازید.
2. از پنل Component Library یک قطعه را انتخاب و Place کنید.
3. روی بوم کلیک کنید تا قطعه واقعاً ساخته شود.
4. روی یکی از Pinها کلیک کنید تا سیم‌کشی شروع شود.
5. روی Pin قطعه دیگر، یک Junction یا یک سیم موجود کلیک کنید تا اتصال کامل شود.
6. برای لغو سیم‌کشی Esc یا راست‌کلیک را بزنید.
7. برای حذف آیتم انتخاب‌شده Delete را بزنید.
