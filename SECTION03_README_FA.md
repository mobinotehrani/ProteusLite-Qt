# بخش سوم — بوم طراحی و محیط کار

این نسخه، خروجی تجمعی بخش‌های ۱ تا ۳ پروژه ProteusPro است. در بخش سوم یک بوم مبتنی بر `QGraphicsView` اضافه شده است که اندازه آن از پروژه جدید یا فایل JSON/XML خوانده می‌شود.

## قابلیت‌های بخش سوم

- رسم Grid با فاصله ۲۰ پیکسل و خطوط اصلی در هر پنج خانه
- نمایش مرز صفحه و مبدأ مختصات
- پیش‌نمایش نقطه Snap زیر نشانگر موس
- نمایش زنده مختصات خام و مختصات Snap‌شده در Status Bar
- Zoom با چرخ موس، دکمه‌های نوار ابزار و میانبرهای صفحه‌کلید
- Panning با دکمه وسط موس یا `Space + Left Drag`
- فعال/غیرفعال کردن Grid و Snap
- بازگشت به زوم ۱۰۰ درصد و Fit کردن کل صفحه
- اعمال اندازه انتخاب‌شده در Start Menu روی Scene

## میانبرها

| عملکرد | میانبر |
|---|---|
| پروژه جدید | `Ctrl+N` |
| باز کردن پروژه | `Ctrl+O` |
| روشن/خاموش کردن Grid | `Ctrl+G` |
| روشن/خاموش کردن Snap | `Ctrl+Shift+G` |
| Zoom In | `Ctrl++` |
| Zoom Out | `Ctrl+-` |
| زوم ۱۰۰ درصد | `Ctrl+0` |
| Fit Sheet | `F` |
| Panning | دکمه وسط موس یا `Space + Drag` |

## نکته معماری

کلاس `CanvasView` تمام رفتارهای مربوط به نمایش بوم را کپسوله می‌کند. متد `snapToGrid` در بخش‌های بعدی برای قرار دادن و جابه‌جایی قطعات و نقاط سیم استفاده می‌شود. در این مرحله، قطعات موجود در فایل فقط شمارش می‌شوند و بازسازی گرافیکی آن‌ها به بخش مدل و تعامل قطعات واگذار شده است.
